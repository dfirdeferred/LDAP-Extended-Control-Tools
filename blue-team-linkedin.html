Purple-team proof · AD LDAP extended controls

A compromised user read the whole directory — your DC logged nothing. One tripwire caught it.

OBJECT_SECURITY DirSync lets any domain user bulk-enumerate objects and group memberships with zero Event 1644 and zero Event 4662. A single SACL audit-canary sees it anyway.

red → blueattack · detection
# attacker: a phished domain user — no admin, no replication rights
j.miller@WKSTN:~$ ldapctl collect --dc dc01.corp.local --user 'CORP\j.miller' --filter '(objectClass=user)'
collected 2686 objects · 348 groups (289 with member lists) -> users.json
footprint: 0 Event 1644 · 0 Event 4662 ← nothing in the DC's default logs
# defender: a SACL canary sits on the crown jewels
PS defender@DC-ADMIN> Test-SaclCanary -Target 'CN=AdminSDHolder,CN=System,DC=corp,DC=local' -Audit Read
Target : CN=AdminSDHolder,CN=System,DC=corp,DC=local
Audit : Read
Fired4662 : True
